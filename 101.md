# Enumeration
  
   1. Network & Service Enumeration
      * `nmap -sV -A <ip-address>`
      * `ident-user-enum 192.168.1.13 22 80 445`<br>
  2. Windows & SMB Enumeration
       * `enum4linux-ng -A <ip-address>`<br>
  3. Web Directory & Subdomain Enumeration
       * `gobuster dir -u http://example.com -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -x php,txt`<br>
       * `gobuster dns -d example.com -w /usr/share/wordlists/subdomains.txt`
       * `nikto -h http://<ip-address>`<br>
  4. DNS Enumeration
       * `dnsenum example.com`
       * `sublist3r -d example.com`<br>
